# README

&gt; **Android 原生智能用药助手 | 端云协同架构 | 离线优先设计**

基于 **Kotlin + Room + Aliyun DashScope** 构建的适老化用药管理应用，专为独居老人及视障人士设计。集成多模态大模型识别、本地药品数据库与智能闹钟提醒，实现"拍药-问药-提醒-追溯"的完整闭环。

## 核心功能特性

### 1. 智能药品识别（AI Vision）
- **多模态大模型**：接入阿里云 DashScope（Qwen-VL），拍照即识药
- **离线兜底**：Room 预置 50+ 常用药基础数据，无网络时仍可基础查询
- **语音交互**：支持 SpeechRecognizer 语音提问与 TTS 语音播报，全程无需阅读

### 2. 用药历史追溯（Data Persistence）
- **自动记录**：每次查询自动写入 SQLite（药品名、图片路径、AI 回答、时间戳）
- **本地隐私**：所有数据存储于 `/data/data/[package]/databases/`，不上传云端
- **周报生成**：基于历史记录统计本周用药频次与药品种类

### 3. 智能闹钟提醒（Medication Reminder）
- **精准定时**：基于 Android AlarmManager 实现系统级闹钟，杀进程仍有效
- **关联药品**：闹钟绑定具体药品与服用剂量，提醒时语音播报用药说明
- **智能重复**：支持按天/周/月周期提醒，适配长期用药（降压药等）与短期疗程（抗生素等）
